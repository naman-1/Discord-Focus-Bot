This discord bot connects with your google Calendar and gives you all the events scheduled on a given date
You can also check out some other commands using --help
https://discord.gg/6KtwApkZM2 -- join this server to test the bot
To run the code you have to run the following commands:
1) pip install discord
2) pip install --upgrade google-api-python-client google-auth-httplib2 google-auth-oauthlib
